## 🛠️ Tecnologias e Conceitos Aplicados
Este projeto foi desenvolvido utilizando as ferramentas e práticas mais modernas do ecossistema Android:

*   **Linguagem:** Kotlin
*   **Interface:** Jetpack Compose (UI Declarativa)
*   **Arquitetura:** MVVM (Model-View-ViewModel)
*   **Componentes:** Navigation Compose, Lifecycle e States.

## 📂 Estrutura do Projeto
*   `/app/src/main/java`: Contém a lógica de negócio, ViewModels e as UI Screens.
*   `/app/src/main/res`: Recursos estáticos (ícones, temas e cores).
